# Supply Chain Analysis and Forecasting
Overview

Python-based data analysis and forecasting system designed to process retail or supply-chain sales data. The project demonstrates how to build a complete data-analytics workflow—from loading raw data, cleaning it, analyzing patterns, and generating insights or forecasts.
This system was developed as part of a DEPI academic project.

-Repository Structure

DEPI_Project.py: Main execution script.

Retail-Supply-Chain-Sales-Dataset.xlsx: Sample dataset used for analysis.

Retail_Final_Project_Data.pbix: Power BI report and visualization output.

DEPI_Final_Project_Report.docx: Detailed documentation and project report.

README.md — Project description and instructions.

Features

Reads and processes structured sales data from Excel.

Performs forecasting or trend analysis on cleaned data.

Exports results for visualization and reporting.

Easy to customize for different datasets and analytics goals.
